# 校园展览馆校友资料系统 · 页面设计文档

## 1. 设计总基调
两套视觉体系：
- 【管理后台】= 专业 · 高效 · 克制（浅色，数据密度高）
- 【大屏展示端】= 科技 · 厚重 · 仪式感（深色，沉浸轮播）

## 2. 视觉规范

### 2.1 字体应用
| 字体 | 用途 |
|---|---|
| HarmonyOS_SansSC_Bold | 中文标题、强调文字、卡片标题 |
| HarmonyOS_SansSC_Regular | 中文正文、列表、表单 |
| din-bold-2 | 大屏数字（年份、统计数据、计数器） |
| Rajdhani-SemiBold-5 | 大屏英文标题、装饰性英文、栏目标签 |

### 2.2 配色

【管理后台】浅色
- 主色 #2B5AED（学府蓝）
- 辅助 #36C5A6（通过/成功）
- 警示 #F5A623（待审）/ #E5484D（驳回）
- 背景 #F5F7FB / 卡片 #FFFFFF
- 文字 #1D2129 / #4E5969 / #86909C

【大屏端】深色科技
- 背景渐变 #0A1326 → #0E1B3A
- 主光色 #1E8BFF（电光蓝）
- 高亮金 #E6B450（荣誉感）
- 玻璃卡片 rgba(255,255,255,0.06) + 1px 边框 rgba(120,170,255,0.25)
- 文字 #EAF2FF / #93A7C9

### 2.3 圆角/间距/动效
- 后台圆角 8px，卡片间距 16px
- 大屏圆角 16px，发光描边，轮播淡入+轻微缩放，时间线流光

## 3. 管理后台页面设计

### 3.1 登录页
左侧校园插画/校徽，右侧登录卡片（账号、密码、角色自动识别）。

### 3.2 工作台
顶部 4 个统计卡（校友总数/待我审核/已上架/本月新增），
中部资料分类占比饼图 + 审核流转柱状图，右侧待办列表。

### 3.3 校友管理
- 列表：搜索栏（姓名/学号/学院/年份）+ 表格 + 分页 + 导入按钮
- 详情：基础信息卡 + 该校友档案条目 Tab（荣誉/作品/成绩）
- 编辑：抽屉表单 + 头像上传

### 3.4 资料档案编辑
左侧表单（标题/分类/时间/描述/绑定校友），
右侧媒体上传区（图片/视频拖拽上传 + 缩略图列表）。

### 3.5 审核中心
- 顶部状态 Tab：待审核 / 已通过 / 已驳回
- 列表显示提交人、学院、分类、提交时间
- 审核详情弹窗：资料预览 + 通过/驳回（驳回必填意见）

### 3.6 资料库运营
- 上架管理：网格卡片，开关上下架，拖拽排序、置顶角标
- 轮播配置：左侧轮播池（可拖入移出）+ 右侧参数（时长/动效/顺序）+ 预览

### 3.7 系统管理
学院字典表、账号表格（角色/学院标签）、日志查询。

## 4. 大屏展示端页面设计（横屏 1920×1080 / 4K）

### 4.1 默认轮播大屏（首页）
- 顶部：校徽 + 标题"XX校园荣誉展览馆" + 实时时间（DIN字体）
- 中央：大幅校友卡轮播（头像、姓名、届别、代表荣誉、作品大图）
- 底部：滚动的校友头像带 / 荣誉关键词流
- 四角科技装饰边框 + 粒子背景

### 4.2 校友墙
网格/瀑布流头像卡，hover 微光，点击进详情。
顶部分类 Tab + 届别筛选条。

### 4.3 校友详情·成长轨迹
左侧大头像 + 基本信息，
右侧垂直时间线（年份用 DIN 大字），
节点展开荣誉/作品/成绩卡片 + 媒体九宫格/视频。

### 4.4 查询页
顶部大搜索框 + 虚拟键盘，下方结果网格。

### 4.5 人脸识别交互浮层
- 待机：右下角"刷脸看专属轨迹"提示 + 摄像头微光
- 识别中：中央扫描环动画 + "正在识别…"
- 命中：人物头像汇聚 + "欢迎，XX！"过场 → 切入该校友专属轨迹轮播
- 未命中：温和提示，3 秒后回默认轮播

## 5. 响应/降级
大屏端网络异常 → 显示本地缓存的上一轮数据 + 角标提示；
摄像头异常 → 隐藏刷脸入口，纯默认轮播。
